# zcj_blog
## 博客系统是基于Hugo
* hugo 源码下载在/src/hugo(也可以在hugo官网下载https://gohugo.io)
* 进入./src/hugo目录下
*  git pull 拉去最新代码
*  go build mian.go 编译
* hugo -v 查看是否安装成功
